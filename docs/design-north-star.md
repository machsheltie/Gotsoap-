# Got Soap? — Design North Star (BINDING — owner-directed, 2026-07-08)

> This outranks the visual-execution guidance in `design.md`. It exists because the first
> build drifted into safe corporate-SaaS layout — the exact opposite of the brief. Every
> redesign decision is checked against the one test below. `design.md`'s *system* (tokens,
> two-author fiction, registers, IA) still stands; its restrained execution defaults do not.

## The one test — applied at EVERY design decision
> **Is this creative, or does it look like a generic corporate SaaS template?**
> **If it reads SaaS/template → AUTO-REJECT. No exceptions.**

## What we are building
An **immersive, totally-committed fictional MOVEMENT that sells the bit.** A visitor must feel
they **stumbled into a real, unhinged hygiene crusade** — not browse a portfolio, not scroll a
landing page. **Immersion over catalogue.** Commit 100%, deadpan, no winking, no safe.

## Why (the stakes)
The site IS the portfolio piece. Its only job: make a creative director at **Liquid Death /
Cards Against Humanity / Dr. Squatch** stop and think *"who made this — I need them."* A site
ChatGPT could generate for a random pooper-scooper startup is, by definition, a failure.

## The bar
Maximalist, editorial, **high-fashion camp**; fragrance-ad drama (Old Spice "smell like a man,"
GQ spreads, Leibovitz milk-mustache, Liquid Death). The five posters are **monumental, immersive,
cinematic artifacts** — full-bleed, environmental — **never boxed thumbnails beside caption text.**
Three distinct dramatic register-worlds (porcelain / smoke / marble). Type **performs** (chrome,
gold, steam-glow, extreme scale). Texture, steam, chrome, marble, foam — **atmosphere everywhere.**

## AUTO-REJECT patterns (if you reach for these, stop)
- Alternating image-left / image-right content rows
- Boxed cards or poster thumbnails sitting beside caption text
- Hero → subhead → centered CTA button
- Outline / pill buttons; evenly-stacked equal-weight sections
- Cookie-banner / toast notifications
- "A gallery of things, each with a little blurb"
- Anything that reads as a SaaS / agency / startup landing template

## KEEP (this is a PRESENTATION redesign, not a rebuild)
The 8 routes / IA · the **frozen copy** (`copy.ts`, owned by the copy lane — design changes only)
· the **two-author fiction** (Campaign 🧼 vs CWAAA 📋, never blended on one surface) ·
**accessibility** (WCAG 2.1 AA, reduced-motion-safe) · the Astro build · **Buttondown** pledge.

## CWAAA surfaces (owner directive)
`/pledge` (Form CW-1) and `/crisis` go **full dramatic certificate / official dossier** — aged
paper, guilloché / security border, watermark seal, letterpress, wax-stamp, ornate rules —
overly-formal bureaucracy played completely straight. (Overrides `design.md`'s "modest, no-texture" CWAAA call.)

## ROUND 2 — sharper mandate (owner + 2nd adversarial review, 2026-07-08)

**The reframe, sharper:** the site is the **active OPERATIONS HUB of a militant movement** — the
visitor breached the aggressively-funded main hub of a deadly-serious hygiene crusade. Two
militant arms, quarantine intact: the **glossy PROPAGANDA arm** (campaign — weaponized
high-fashion fragrance-ad billboards) and the **bureaucratic ENFORCEMENT arm** (CWAAA —
terrifyingly formal, full-viewport government decrees). The register: **militant propaganda
manifesto × over-the-top high-fashion editorial.** If a section reads "tasteful / clean /
system," it is WRONG — go louder, heavier, more committed.

**Newly-named AUTO-REJECT tells (added to the list above):**
- **Thin architectural / graph-paper grid backgrounds** — the AI-dev-tool / crypto-analytics /
  tax-software signature. Bathroom TILE is allowed ONLY if it reads as *wet glossy ceramic*
  (big tiles, grout, specular highlights, condensation, steam) — a **material**, never a fine line-grid.
- **A standard navbar** (logo-left / links-right / hamburger `[MENU]`). It reads as a checkout flow.
- **"SPOT NO. n" ad-showcase labels.** The posters are the movement's documented
  **OFFENSES / FIELD INTERVENTIONS / PUBLIC HAZARDS**, not "spots."
- **Floating soft-drop-shadow document cards** (invoice / PDF / dashboard look).
- **Any mid-scroll apology for the bit.** The experience NEVER breaks character mid-page.
- **Orderly alternating cream/gray blocks in a tidy sequence.**

**Binding directives:**
- **Weaponize the bureaucracy:** the Declaration (pledge) and the Memorandum/dossier (crisis)
  each occupy the **FULL VIEWPORT** — brutalist borders, stark margins, terrifyingly formal
  decree typography. Full-screen official documents, never web forms or centered cards.
- **Relabel the posters** as OFFENSES / FIELD INTERVENTIONS / PUBLIC HAZARDS (institutional).
  *(Copy-lane polishes the exact wording; design uses institutional placeholders meanwhile.)*
- **Kill the navbar → build an official masthead / emergency-broadcast warning bar** (over-designed, aggressive).
- **Bury the disclaimer:** "None of these men are real" lives ONLY as microscopic footer fine
  print. The reveal is `/about` (the page-three punchline), never a mid-home confession.
- **Sections collide, overlap, and break their containers;** type is heavy and demanding; steam
  clouds bleed across edges. No orderly stack of centered blocks.
